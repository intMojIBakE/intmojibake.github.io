2. 安装 Minecraft

2.1 下载启动器

Linux 下可用的启动器不多, 也不建议用 wine 跑 Windows 下的启动器. 正版用户可以在官网下载 Linux 下的启动器. 其他用户可以下载 HCML 启动器.
现在已经可以直接进行游戏, 在启动器目录打开终端, 输入 java -jar ***.jar & 即可, 其中 ***.jar 是你的启动器的文件名.
